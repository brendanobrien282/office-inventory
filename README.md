# Office Inventory

A simple local inventory system for tracking snacks and drinks in a small office.

## Quick start

Run each in a separate terminal:

```bash
cd server && npm install && npm run dev   # API on :3001
cd client && npm install && npm run dev   # UI on :5173
```

Open http://localhost:5173

## Tests

```bash
cd server && npm test
cd client && npm test
```
